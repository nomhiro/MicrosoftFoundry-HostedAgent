# sales-staff-agent ログ出力機能

## 実装日
2026-02-01

## 概要
sales-staff-agentにツール呼び出しとLLM呼び出しのログ出力機能を追加。

## 実装内容

### container.py の変更点
1. `logging`モジュールを使用したロギング設定を追加
2. `responses.create(stream=True)`でストリーミングモードを使用
3. イベントタイプに応じたログ出力:
   - `response.created`: レスポンスID
   - `response.output_item.done`: ツール呼び出し (mcp_tool_call) / 結果 (mcp_tool_call_output) / メッセージ
   - `response.completed`: 最終出力
